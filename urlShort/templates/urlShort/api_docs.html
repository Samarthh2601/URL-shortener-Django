API Documentation

The API can be accessed through a "POST" request to the route "/api" which requires a "long_url" parameter to be included in the headers.

For example: headers = {"long_url": "http://www.google.com"}

Here "long_url" is the URL you want to shorten.

Example Response:

{
        'short_url': new_short_url,
        'long_url': original_url,
        'clicks': number_of_times_used,
        'created_at': link_created_at,
    }
